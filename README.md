🎬 Movie Search App

A responsive Movie Search App built using HTML, CSS, and JavaScript that allows users to search for movies and view details such as posters, titles, and release years.

The application fetches movie data from the OMDb API, providing real-time search results through a clean and interactive interface.

🚀 Features

* Search movies by title
* Display movie posters
* Show movie title and release year
* Fetch data using OMDb API
* Responsive and simple user interface

🛠 Technologies Used

* HTML
* CSS
* JavaScript
* OMDb API

🎯 How to Use

1. Enter the movie name in the search box
2. Click the Search button
3. View the movie results with poster and details
